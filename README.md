# projeto-contatos
Projeto acadêmico para criar um sistema de "Agenda de Contatos" capaz de armazenar e gerenciar dados pessoais (nome, telefone, e-mail). O desenvolvimento é incremental, evoluindo de um cadastro simples para uma aplicação robusta com validação de dados, busca e armazenamento múltiplo usando arrays e structs, e modularizado em funções.
